# ♟️ AIMate — AI-Based Smart Chess Board

A smart chess game with an AI opponent, implemented in **Python** and **Java**. The AI uses classic adversarial search algorithms to play challenging chess at various difficulty levels.

---

## 🧠 AI Algorithms Used

- **Minimax Algorithm** — evaluates all possible moves up to a given depth
- **Alpha-Beta Pruning** — optimizes Minimax by eliminating unnecessary branches
- **Negamax Algorithm** — a simplified variant of Minimax for two-player zero-sum games

## 🚀 Getting Started

### Python (AI Engine)

**Requirements:**
- Python 3.x
- `pygame` library

**Install dependencies:**
```bash
pip install pygame
```

**Run the game:**
```bash
cd "Artificial Intelligence"
python ChessMain.py
```

### Java (Networked Chess)

**Requirements:**
- Java JDK 8 or higher

**Compile and run:**
```bash
cd "Computer Networking/Chess-main"
javac *.java
java Main
```

## 🎮 Features

- Play against an AI opponent powered by Minimax + Alpha-Beta Pruning
- Two-player local and networked chess (Java module)
- Full chess rule implementation (castling, en passant, pawn promotion)
- Visual chess board rendered with Pygame (Python) / Java Swing (Java)
- Adjustable AI search depth for difficulty control
